# Drift

> Reactive gaming server

## Building

To develop:
```
./mvnw compile quarkus:dev
```

To test:
```
./mvnw clean test
```

To build:
```
./mvnw clean package -Pnative
```

To run:
```
./target/drift-1.0.0-SNAPSHOT-runner
```

## Made with

- Kotlin
- Vert.X
- Quarkus.io
